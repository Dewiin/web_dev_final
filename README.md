## Installation & Setup

### Prerequisites
Ensure you have Node.js and npm installed:
```sh
node -v  # Should be 18+  
npm -v
```

### Clone the Repository
```sh
git clone https://github.com/Dewiin/web_dev_final.git
cd web_dev_final
```

### Install Dependencies
```sh
npm install
```

### Run the App Locally
```sh
npm run dev
```
<p align="right"><a href="#readme-top">Back to top</a></p>

## Deployment (Firebase)
Build the project:
```sh
npm run build
```
Login to firebase:
```sh
firebase login
```
Initialize the project and deploy:
```sh
npm run deploy
```
